TextViewStack

public class TextViewStack: TextStack, KeyboardResponderInputContainer

TextStack subclass using a UITextView.

placeholderLabel is configured in response to UITextViewTextDidBeginEditingNotification, UITextViewTextDidChangeNotification and UITextViewTextDidEndEditingNotification, which configure the placeholder and underline.

    placeholderLabel

    The UILabel or subclass to show the placeholderText when the user has already entered text. Default value is a new UILabel.
    underline

    The UIView or subclass to use as an underline for textComponent.
    errorLabel

    The UILabel or subclass to use to show the error text. Default value is a new UILabel. The font for this labels is set to UIFontTextStyleCaption2.
