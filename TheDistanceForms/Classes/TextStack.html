TextStack

public class TextStack: ErrorStack

ErrorStack subclass to contain the common elements and logic shared between TextFieldStack and TextViewStack. These show an editable text component with a placeholder, icon, optional error label and error image.

    Parameters

    textComponent

    The view the user will interact with.
    placeholderLabel

    The UILabel or subclass to show the placeholderText when the user has already entered text. Default value is a new UILabel.
    underline

    The UIView or subclass to use as an underline for textComponent.
    errorLabel

    The UILabel or subclass to use to show the error text. Default value is a new UILabel. The font for this labels is set to UIFontTextStyleCaption2.
    iconImageView

    The UIImageView or subclass to use to show the icon. Default value is a new UIImageView.
    errorImageView

    The UIImageView or subclass to use to show the error icon. Default value is a new UIImageView.
